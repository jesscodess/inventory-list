Simple application that lists out items (objects) stored in a JSON server (mock API) and displays them as an inventory list.

Project made with the guidance of [this tutorial](https://www.youtube.com/watch?v=jKyAEj0EvAA) (repo can be found [here](https://github.com/atapas/youtube/tree/main/react/27-inventory-useReducer-jsonserver-axios)) by Tapas Adhikary, published in his YouTube channel.

Scaffolded with Vite, it utilizes React, node.js, Tailwind CSS, axios and json-server.

Aplicação simples que lista itens (objetos) armazenados em um servidor JSON (API simulada) e os exibe em uma lista de inventário.

Projeto feito com a orientação [deste tutorial](https://www.youtube.com/watch?v=jKyAEj0EvAA) (o repositório se encontra [aqui](https://github.com/atapas/youtube/tree/main/react/27-inventory-useReducer-jsonserver-axios)) por Tapas Adhikary, publicado em seu canal no YouTube.

Feito com Vite, React, node.js, Tailwind CSS, axios e json-server.
